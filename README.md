## Getting Started

### Prerequisites

  - npm ( Node Package Manager )
  - git and Github Account

### Setup

  - Open terminal
  - Change the directory then run the following:
    - `git clone ggit@github.com:estebanmual/covid-tracker.git`
  - cd repo-name/

### Install

  - Run  `npm install` on the root directory of your project
